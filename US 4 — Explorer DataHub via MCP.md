# US4 — Explorer DataHub via MCP

En tant que développeur,  
je souhaite utiliser un assistant IA pour explorer DataHub via MCP,  
afin de répondre à plusieurs questions métier sans connaître l’API.

---

## WHY

L’objectif est de valider que MCP permet une exploration réelle et efficace de l’API DataHub.

---

## WHAT

Résoudre les tâches suivantes via MCP :
1. Rechercher un document lié à un mot-clé donné
2. Ouvrir un document pertinent
3. Extraire une information précise depuis ce document

---

## HOW

- Travaille dans `mcp/exercises/exo4/`
- Utilise tools et resources MCP disponibles
- Essaie différentes formulations de requêtes si nécessaire
- Observe le comportement de l’assistant IA

---

## RESSOURCES

- Serveur MCP fonctionnel
- Historique des tools/resources exposés

---

## VALIDATION CRITERIA

- Les 3 tâches sont résolues via MCP
- L’assistant utilise effectivement tools et resources
- Les réponses sont cohérentes avec les données DataHub
